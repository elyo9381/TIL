## 선택 정렬

* 문제 
N개이 숫자가 입력되면 오름차순으로 정렬하여 출력하는 프로그램을 작성하세요.
정렬하는 방법은 선택정렬입니다.

* 알고리즘
idx를 기준으로 j가 j는 i+1값이다. 반복문을 돌게 되는데 j의반복문에서 최소값을 찾아서 j의 인덱스를 idx의 인덱스로 옮긴다. 
그리고 나서 tmp를 이용하여 교환한다. 


```
#include <stdio.h>
#include <algorithm>
#include <vector>
using namespace std;

int main() { 
    int n ,tmp;
    vector<int> a(n);

    scanf("%d",&n);

    for(int i = 0; i<n; i++){ 
        scanf("%d", &a[i]);
    }

    for(int i = 0 ; i<n-1 ;i++) {
        int idx = i;
        for(int j = i+1; j<n; j++) { 
            if(a[idx] > a[j] ){
                idx = j;
            }
            tmp = a[i];
            a[i] = a[idx];
            a[idx] = tmp;
        }
    }
    for(int i = 0; i<n; i++) {
        printf("%d ", a[i]);
    }
}
```
