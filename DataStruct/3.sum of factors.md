## 약수의합

자연수 n이 주어지면 자연수 N의 진약수의 합을 수식과 함꼐 추렭하는 프로그램을 작성하세요.

약수라는 것은 N이 주어졌을때 나눠떨어지는 수로 MOD를 사용하여서 나눠떨어지는 수를 의미한다.

~~~
#include <iostream>

using namespace std;

int main(){
	int n, sum = 1;

	cin >> n;

	cout << "1";

	for(int i =2; i<n; i++)
	{
		if(n%i == 0) {
			cout << " + " << i; sum +=i;
		}
	}
	cout << " = " <<sum;

	return 0;
}
~~~
