## 재귀함수(스택)를 이용한 2진수 출력

* 문제 

이진트리 전위순회와 후위 순회를 연습해보지

* 알고리즘

전위순회는 부모먼저 그리고 왼쪽자식 오른쪽자식이고
중위순회는 왼쪽자식 부모출력 오른쪽 자식순으로 진행되고
후휘순회는 왼쪽자식 오른쪽자식 다끝내고 출력하는 순서이다. 

```
#include<cstring>
#include<vector>
#include<algorithm>
#include<stack>
using namespace std;


int recur(int v){
    if(v > 7) return 0; // 경우의수나 완전탐색을 할때 최종목적지에 왔을때
    else {
        // 전위순회
        recur(v*2);
        // 중위순회
        recur(v*2+1);
        // 후위순회
        printf("%d ",v);
    }
    return 0;
}
int main(){
    recur(1);   

    return 0;
}
```
