# 이커머스

- 인터파크 티켓과 같이 티켓팅에 특화된 쇼핑몰

- 주기적인 대기열 시스템 사용

- ex) IU 티켓팅 : 6만석 가량의 티켓을 티켓을 판매하는데, 34만명 이상이 몰려들었다.
